### 2. 重写 `discover.html` — 发现页交互全面上线（549 行）

**上一版本问题：** 纯静态页面，所有按钮（搜索栏、筛选、Tab、加入、底部导航）均无事件绑定。

**本版本修复/新增：**

| 控件 | 功能 |
|------|------|
| 🔍 搜索栏 | `input` 事件实时过滤活动列表（匹配标题/类型/地点） |
| 🎛️ 筛选按钮 | `prompt` 弹窗按活动类型筛选，支持 10 种类型 |
| 📑 Tab 切换 | 推荐 / 附近 / 同校 / 最新 — 点击切换高亮并刷新列表 |
| ➕ 加入按钮 | `POST /api/activities/:id/join`，自动处理 4 种按钮状态 |
| 🧭 底部导航 | 跳转发现/活动/消息/我的 四个页面 |
| 🔐 登录检查 | 无 token 自动重定向到 `/login.html` |
| 🏗️ 骨架屏 | 加载中显示灰色占位卡片 |
| 🎨 类型图标 | 11 种活动类型各有独立渐变色背景 + Font Awesome 图标 |
| 📭 空态提示 | 无搜索结果或加载失败时显示提示文字 |
| 🃏 卡片点击 | 点击活动卡片弹出详情提示（预留详情页入口） |

**加入按钮 4 种状态：**
1. **可加入** — 黑底按钮，显示 `+剩余名额 加入`
2. **已加入** — 绿色按钮，显示 `已加入 ✓`（disabled）
3. **已满员** — 灰色按钮，显示 `已满员`（disabled）
4. **我创建的** — 橙色按钮，显示 `我创建的`（disabled）

**11 种活动类型图标映射：**
徒步 / 骑行 / 露营 / 跑步 / 篮球 / 足球 / 羽毛球 / 桌游 / 自习 / 聚餐 / 其他
